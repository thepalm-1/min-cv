Kort forklart

Denne CV-siden viser offentlig yrkesinformasjon om Lars Palm. Du fyller ikke inn eller laster opp personopplysninger på siden, og Palmtech mottar ikke opplysninger om deg gjennom CV-funksjonaliteten.

Hvem er ansvarlig?

Palmtech er ansvarlig for nettstedet. Spørsmål om personvern kan meldes via prosjektets offentlige GitHub-side.

Opplysninger som vises i tjenesten

Nettstedet publiserer CV-opplysninger om sidens eier, blant annet navn, kontaktinformasjon, utdanning, arbeidserfaring og teknisk kompetanse.

Formålet er å presentere en profesjonell CV og gjøre en PDF-versjon tilgjengelig for nedlasting.

Hvordan opplysninger behandles

  • CV-innholdet er en del av det offentlig publiserte nettstedet.
  • Når du velger å laste ned PDF, genereres filen lokalt i nettleseren fra det samme publiserte CV-innholdet.
  • Lenker til for eksempel GitHub og LinkedIn tar deg videre til eksterne tjenester med egne personvernregler.

Lagring, cookies og statistikk

Nettstedet bruker ikke cookies eller analyseverktøy. Valgt språk og lyst eller mørkt tema lagres i nettleserens lokale lagring (localStorage) for å huske innstillingene dine ved senere besøk. Disse innstillingene inneholder ikke CV-innhold eller kontaktopplysninger om deg.

Publisering av nettstedet

Nettstedet publiseres som en statisk side gjennom GitHub Pages. GitHub kan behandle tekniske opplysninger om besøk til nettstedet, for eksempel IP-adresse og forespørselslogger, i samsvar med GitHubs personvernerklæring.

Dine valg og rettigheter

Du kan slette lagrede språk- og temavalg gjennom innstillingene for nettstedsdata i nettleseren din. Nettstedet ber deg ikke om å oppgi personopplysninger.

Dersom du mener at behandling av personopplysninger ikke er i tråd med personvernreglene, kan du kontakte Palmtech eller klage til Datatilsynet.
